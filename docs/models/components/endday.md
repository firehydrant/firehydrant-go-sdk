# EndDay

The day of the week on which the restriction should end, as its long-form name (e.g. "monday", "tuesday", etc).


## Values

| Name              | Value             |
| ----------------- | ----------------- |
| `EndDayMonday`    | monday            |
| `EndDayTuesday`   | tuesday           |
| `EndDayWednesday` | wednesday         |
| `EndDayThursday`  | thursday          |
| `EndDayFriday`    | friday            |
| `EndDaySaturday`  | saturday          |
| `EndDaySunday`    | sunday            |
