# ServiceTier

Integer representing service tier. Lower values represent higher criticality. If not specified the default value will be 5.


## Values

| Name               | Value              |
| ------------------ | ------------------ |
| `ServiceTierZero`  | 0                  |
| `ServiceTierOne`   | 1                  |
| `ServiceTierTwo`   | 2                  |
| `ServiceTierThree` | 3                  |
| `ServiceTierFour`  | 4                  |
| `ServiceTierFive`  | 5                  |
