# ExactLevel

Returns log entries of all levels equal to the provided level


## Values

| Name                | Value               |
| ------------------- | ------------------- |
| `ExactLevelUnknown` | unknown             |
| `ExactLevelDebug`   | debug               |
| `ExactLevelInfo`    | info                |
| `ExactLevelWarn`    | warn                |
| `ExactLevelError`   | error               |
| `ExactLevelFatal`   | fatal               |
