# OrderBy

Sort runbooks by their updated date or name. Accepts 'updated_at', 'name', 'owner', 'last_executed_at', and 'created_at'.


## Values

| Name                    | Value                   |
| ----------------------- | ----------------------- |
| `OrderByUpdatedAt`      | updated_at              |
| `OrderByName`           | name                    |
| `OrderByCreatedAt`      | created_at              |
| `OrderByLastExecutedAt` | last_executed_at        |
| `OrderByOwner`          | owner                   |
