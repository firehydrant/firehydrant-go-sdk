# OfLevel

Returns logs of all levels equal to or above the provided level


## Values

| Name             | Value            |
| ---------------- | ---------------- |
| `OfLevelUnknown` | unknown          |
| `OfLevelDebug`   | debug            |
| `OfLevelInfo`    | info             |
| `OfLevelWarn`    | warn             |
| `OfLevelError`   | error            |
| `OfLevelFatal`   | fatal            |
