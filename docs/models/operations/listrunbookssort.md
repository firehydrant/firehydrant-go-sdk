# ListRunbooksSort

Sort runbooks by their updated date. Accepts 'asc', 'desc'. This parameter is deprecated in favor of 'order_by' and 'order_direction'.


## Values

| Name                   | Value                  |
| ---------------------- | ---------------------- |
| `ListRunbooksSortAsc`  | asc                    |
| `ListRunbooksSortDesc` | desc                   |
